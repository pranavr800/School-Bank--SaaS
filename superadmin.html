Setup Guide
How this portal is secured and how to configure a developer account
Initial Setup — Firestore superAdmins Collection
1

Find your Firebase UID

Firebase Console → Authentication → Users → copy UID.

2

Create the superAdmins collection

Firestore → Start Collection → ID: superAdmins → Document ID = your UID → Field: role = superAdmin

3

Add AI Assistant key

Go to console.groq.com/keys, get a free Groq key, then open ⚙️ Settings in the topbar and paste it. Saved to Firestore — works everywhere.

4

Deploy to Netlify

Drag and drop this HTML file to Netlify as a separate site. Use a non-obvious URL slug.
